## How to Run the Game

## How to Run on Windows

1. Open the chess engine folder in **VS Code**.

2. Open the terminal by pressing:

```text
Ctrl + `
```

3. Copy and paste this command, then press **Enter**:

```powershell
py -m pip install pygame
```

4. After that finishes, copy and paste this command, then press **Enter**:

```powershell
py src/main.py
```

The game should now open.

If it says `No such file or directory`, make sure your terminal is inside the main chess engine folder, the same folder that contains the `src` folder.


### macOS

1. Open the folder in **VS Code**.

2. At the top of VS Code, click:

```text
Terminal → New Terminal or press Control + `
```

3. Copy and paste this command, then press **Enter**:

```bash
python3 -m pip install -r requirements.txt
```

4. After that finishes, copy and paste this command, then press **Enter**:

```bash
python3 src/main.py
```

The game should now open.

---

### If the Game Does Not Open

Make sure you are inside the correct project folder.

The folder should be called:

```text
Peter chess engine v6.3.1
```

Also make sure Python is installed on your computer.
