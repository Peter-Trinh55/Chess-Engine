## How to Run the Game

### Windows Instructions

1. Open **VS Code**.

2. Click:

```text
File → Open Folder
```

3. Select the folder called:

```text
Peter Chess Engine v6.3.1 Copy
```

Make sure you open this specific folder, not the outer Downloads folder.

4. Open the terminal by pressing:

```text
Ctrl + `
```

The ` key is usually under the Esc key.

5. Copy and paste this command into the terminal, then press **Enter**:

```powershell
py -m pip install pygame
```

6. Wait until pygame finishes installing.

7. On the left side of VS Code, find the folder called:

```text
src
```

8. Click the arrow next to `src` to open the dropdown.

9. Click:

```text
main.py
```

10. Press the **Run** button in the top-right corner of VS Code.

11. Enjoy playing the game.
