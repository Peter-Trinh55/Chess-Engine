## How to Run the Game

### Pre-requisites

Before running the game, make sure you have:

* **Python installed**
  Recommended version: **Python 3.12**

* **Visual Studio Code installed**
  This is used to open and run the chess engine project.

---

### How to Download the Program

1. Go to the GitHub page for the chess engine.

2. Press the green button called:

```text
Code
```

3. Press:

```text
Download ZIP
```

4. Wait for the ZIP file to download.

5. Go to your **Downloads** folder.

6. Right-click the ZIP file and press:

```text
Extract All
```

7. Open the extracted folder.

8. Make sure you open the specific folder called:

```text
Peter Chess Engine v6.3.1 Copy
```

This is the folder you need to open in VS Code.

---

### Windows Instructions

1. Open **VS Code**.

2. Click:

```text
File → Open Folder
```

3. Select the folder called:

```text
Peter Chess Engine v6.3.1 
```

Make sure you open this specific folder, not the outer Downloads folder.

4. Open the terminal by pressing:

```text
Ctrl + `
```

The ` key is usually under the Esc key.

5. Copy and paste this command into the terminal, then press **Enter**:

```powershell
py -m pip install pygame
```

6. Wait until pygame finishes installing.

7. On the left side of VS Code, find the folder called:

```text
src
```

8. Click the arrow next to `src` to open the dropdown.

9. Click:

```text
main.py
```

10. Press the **Run** button in the top-right corner of VS Code.

11. Enjoy playing the game.
